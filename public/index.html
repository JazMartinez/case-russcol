    задача:

    разработка сайта для оптовой и розничной продажи картин профессиональных современных художников «Русская коллекция»

    адрес сайта:

    https://russiancollection.ru/

    цена:

    198 000

    тариф:

    «Интернет-магазин» + доп. работы

    Подготовительные этапы

    Разработка сайта представляет собой комплексный процесс. До начала технических работ было выполнено следующее:

    Семантическое проектирование

    подобрали ключевые слова для продвижения, разработали структуру сайта

    Индивидуальный дизайн

    на основе wireframe и написанных текстов разработали дизайн для главной страницы, каталога и карточки товара

    Wireframe

    разработали прототип для

    1. главной страницы
    2. страницы каталога
    3. карточки товара

    Копирайтинг

    написали текст для главной страницы и для 15 внутренних страниц

    Вёрстка

    По макетам сверстали кроссбраузерные
    адаптивные страницы

    Настройка cms

    включает в себя установку cms на хостинг, перенос верстки
    и реализацию спроектированного функционала

    • Для решения поставленной задачи была выбрана cms bitrix, редакция «Малый бизнес»
    • Потребовалось сменить хостинг, поскольку текущий не удовлетворял требованиям bitrix

    Настройка cms

    Общий шаблон сайта, главная страница

    установили bitrix
    на хостинг timeweb.com

    создали общий шаблон
    и структуру сайта

    перенесли верстку главной страницы, настроили компоненты вывода меню и слайдера

    дальше

    Настройка cms

    Каталог товаров

    настроили фильтрацию по жанрам

    создали разделы с описанием

    настроили общий вид каталога, вывод товаров из разделов

    дальше

    Настройка cms

    Карточка товара

    Добавили возможность
    отобразить картину в 3D

    Для картин на бумаге есть возможность выбора багета (новый багет добавляется в панели администрирования, в карточке отображаются в виде характеристик товара)

    Добавили вНастроили торговые предложения с автоматической генерацией (всевозможные вариации товара в зависимости от характеристик)

    в карточке выводятся характеристики товара (артикул, художник, жанр)

    SEO оптимизация

    Провели базовые работы по оптимизации:

    • настроили протокол https
    • настроили редирект с www.russiancollection.ru на russiancollection.ru
    • создали карту сайта (sitemap.xml)
    • добавили файл robots.txt
    • создали страницу 404 ошибки
    • настроили Яндекс.Вебмастер
    • добавили счетчик Яндекс.Метрики и настроили цели для отслеживания конверсии
    • в формы обратной связи добавили согласие на обработку персональных данных

    В итоге наш клиент получил не только красивый и удобный сайт, но и оптимизированный для поисковых систем.
